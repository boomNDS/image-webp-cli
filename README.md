# image-webp-cli

Batch-convert JPG/PNG images to WebP using `cwebp`.

## Features

- Converts JPEG & PNG (or both) to WebP  
- Set quality `-q 0–100`  
- Preserve directory structure  

## Installation

```bash

```

## Usage

```bash
image-webp -i ./photos -o ./webp-output -t jpg -q 85
```
Run image-webp --help for details.
